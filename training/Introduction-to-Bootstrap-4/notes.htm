Introduction to Bootstrap 4

Lesson 1

  • Added individual bootstrap-grid-min.css in addition to stand bootstrap .css
  • Documentation has great components list with all HTML that can be simply copied and pasted
  • Any type of link or button can be styled to
  • Panels are now called Cards
  • Bootstrap 4 now has Carousels with all the features
  • sm, md, lg, xl still used:
    - breakpoint-xs: 0;
    - breakpoint-sm: 576px;
    - breakpoint-md: 768px;
    - breakpoint-lg: 992px;
    - breakpoint-xl: 1200px;
  • Have to use .container or .container-fluid
  • Do not put content in a row
  • Column default to 12 at xs
  • If you must maintain columns use .col-6, .col-4, etc.
  • You can nest row in columns
