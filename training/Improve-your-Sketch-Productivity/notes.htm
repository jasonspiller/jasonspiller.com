Improve your Sketch Productivity

Lesson 1: Design Essentials with Sketch

  • Toolbar is completely customizable
  • Right bar is the inspector with modifiable values
  • You can create styles, apply them to object and change all at the same time
  • You can export styles as code
  • Symbols are a great way to organzie resuable components
  • Symbols only have to be edited once across pages
  • Inspector has very granular control over values
  • Supports blending modes, patterns (subtlepatterns.com)
  • You can do math right in the value boxes
  • Has gaussian, motion, zoom and background blurs
  • Use pixel fit web work
  • When you create and use symbols, you can exclude text
  • You can add multiple CSS values for borders use T/R/B/L
  • Keyboard shortcuts are controlled by System Preferences > Keyboard > Shortcuts > App Shortcuts
  • Artboards can be used to show different layouts
  • You can also use artboards to show versions
  • You can create a guidelines or a style guide that has the templates for all of your features on a separate page
  • You can use Sketch Measure to add annotations (measurements) on your document

Lesson 2: Bytes: 1 Minute Lessons on Sketch

  • sketchshortcuts.com for all for all keyboard shortcuts
  • You can resize artboards to fit contents
  • Esc will move you up to the parent symbol, page, etc.
  • Smart Guides help show distances
  • When using shared styles make sure to press the sync button
  • You can send symbols directly to the Symbols page
