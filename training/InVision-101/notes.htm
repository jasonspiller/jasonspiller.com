InVision 101

Prototype Basics

  • You need to create screens in another app (Sketch, PS, AI, etc.)
  • You can toggle between project view, preview, build and comment modes
  • Use the cog to set standard settings

Interactive Prototypes

  • Power lies in interactivity
  • Enter Build mode and add hot spots
  • You can set a header or footer to be fixe
  • Overlay screens can act as drop downs with hover or click (you can apply animations)
  • You can include hot spots in a template and rapidly apply to all screens
  • On Mobile project you can add animated screen wipes
  • You can also add specific gestures and what they should do
  • There are also timed screens

Feedback and Collaboration

  • You can collaborate with other people simply with a link
  • There are ways to lock down everything
  • Users can add comments or sketches
  • You can also Live Share and do realtime reviews

Managing Projects in Workflow

  • You can screen progress (On Hold, For Review, etc)
  • You can add due dates, tasks and comments to screens
  • Comments can be converted to tasks
  • Notifications show recent updates

Using Overlay Screens

  • Overlays should be separate graphic elements and screens
  • They can positioned as you see fit
  • They have their own animations
  • Add hot spots directly to the overlay screen

Creating a Board

  • You can any type of assets (fonts, graphics, colors, etc.)
  • You add a header
  • You can arrange as you see fit or choose from auto-arrange options
  • Boards can also be collaborative
  • You click and drag or do click selections

Using GIFs for Custom Interactions

  • Animated GIFs can be added and triggered with hot spots to add flava

Transparent Navigation

  • Add tranparency in your graphics program and InVision will use it

Using LiveShare

  • You can use Skype in conjunction or LiveShare offers a VOIP option

Creating Fixed Elements

  • Add elements to the top or bottom of your design file (with transparent bg if need)
  • Create fixed header or footer from the portion

Replacing Screens

  • You can use the same filenames to replace existing screens
  • Use desktop/cloud syncing

Changning Device Skins

  • In a mobile project select the cog and select skin
  • Does not show up when previewing it on the device

Sharing a Selection of Screens

  • You selection individual screens or groups
  • Hot spots are disabled for screens that not shared

Prototype Organization

  • Use sections to organize screens
  • List view can be convenient
  • You also have the option to use a logical sort, but it will wipe out your section

Retina Sized Images

  • Mobile projects automatically size images
  • Just add @2x to screens and InVision will scale them appropriately
