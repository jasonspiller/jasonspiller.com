Building Your First Website with Bootstrap 4

Lesson 1:

  • Container-Fluid now has padding that has to be removed. Add .no-padding{padding-left:0; padding-right:0}. You must do something similar to remove margin from .row
  • Bootstrap has 3 color schemes built in (Light, Dark and Blue)
  • .bg-inverse for Dark version
  • Bootstrap has many components that can be copied into the project
  • Nav is common one
  • Font-Awesome no substitutes icons with SVG via JS
  • You can float the nav to the right with .mlauto
  • You can align a dropdown to the right of the parent element by adding .dropdown-menu-right to .dropdown-menu
  • jumbotron is a pre-built hero component
  • My observation - no need to have smaller col- size if not needed. Instructor kept putting col-sm-12. Only need modification for large screens (mobile first)
  • Carousel component is not responsive (stretches photo)
  • I added background to navigation arrows and slide captions (.carousel-caption {background: rgba(0,0,0,0.75);}) for better contrast
  • .d-none .d-md-block can hids and show things at the respective size
  • Breakpoints for Bootstrap 4
      @media (min-width: 576px) { ... }
    • @media (min-width: 768px) { ... }
    • @media (min-width: 992px) { ... }
    • @media (min-width: 1200px) { ... }
  • Insert .sr-only usually in span to have screen reader read the text
  • Insert aria-hidden="true" to elements to hide them from screen readers
  • .img-fluid to add responsiveness to images
  • Power lies in nesting rows
  • Add modal at the bottom of all your mark-up
  • You can call a modal by a class or ID
  • There are sm, md and lg modals
  • You can add responsive classes to embedded items (i.e. containing div .embed-responsive .embed-responsive-16by9 and embedded item .embed-responsive-item)
  • For inline items you can put a wrapper around them and use margin: auto to vertically center them
